Biographical and Evaluation Database Management System

Final Year Project - Sheikh Zayed University, Faculty of Computer Science, Information System Department

Project Overview

The Biographical and Evaluation Database Management System is a comprehensive solution designed to streamline human resources management. Our secure and automated system efficiently manages employee biographical data, administrative records, and performance evaluations. Developed using cutting-edge technologies including C#, Visual Studio, SQL Server, and Crystal Report, the system enables detailed record-keeping, automated reporting, and simplified data analysis.

Featuring an intuitive, user-friendly interface enhanced with Pichon Icons (Icons8), the system incorporates UML diagrams to clearly define its structure and workflows. This makes it an ideal solution for organizations seeking integrated, efficient human resources management with robust reporting capabilities.
